Für die Ausführung der Applikation sind folgende Schritte zu tätigen:

1. Aufsetzen der Datenbank bimap mit entsprechenden Tabellen und Datensätze
Nutze hierfür das File CreateBimapDatabase.sql

2. Festlegen deiner Datenbank Username und Password
Im Main.java (ab Zeile 10) sind hierfür zwei Strings vorgegeben, diese sind vorerst auf "root" und "" gesetzt. 

3. Server starten über Main.java

4. Index.html im package bimclient mit Browser starten
Achtung, der index.html muss über den Localhost: gestartet werden, damit das API von geo.admin funktioniert.
Zudem wird ein Starten per debugging von dem API3.admin.geo nicht unterstützt!

Viel Spass beim Nutzen der App! Über allfällige Inputs jeglicher Art sind wir dankbar.
